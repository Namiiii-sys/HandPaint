# Hand Gesture Drawing App

## About
A small experimental project built to explore webcam-based hand gesture recognition using HandTrack.js. The application allows users to draw on a canvas using only hand gestures, combining creative interaction with computer vision in a simple, browser-based setup.

---

## Features

- Webcam-based hand gesture detection  
- Gesture-controlled drawing without mouse or touch input  
- Canvas clearing using open-palm gesture  
- Color changes triggered by fist gestures  
- Lightweight, client-side implementation  

---

## Tech Stack

- Next.js  
- HandTrack.js  
- HTML Canvas API  
- Web Camera API  

---

## Gesture Controls

- Pointing gesture: Draw on canvas  
- Open palm: Clear canvas  
- Closed fist: Change drawing color  

---

## Disclaimer

This is a fun side project built for experimentation and learning, not intended as a production-ready application.
